Git is an online environmrnt which we can send our codes in there because of these features:
1- We can undo our changes
2- A number of programmers can make changes on it
3- Can make web pages everyone can see

after clicking on source control, choose initialize Repository

after installing git, type these in command prompt:
git config --global user.name "Menooa Ebrahimi"
git config --global user.email "menooa.ebrahimi@gmail.com"

after making changes, first commit (with a message), then push, in order to send it to your git account.
